# ❤️‍🔥 Destined — Dating App

**Stop swiping. Start connecting.**

Destined is a modern dating app that helps people find real connections with real chemistry.

---

## 🚀 Features

- 🔥 Swipe & Match
- 💬 Real-time Chat
- 🎁 Gifts & Diamonds
- 🐾 Virtual Pets
- 🏙️ Vibe City
- 👑 Tiers (Free, Premium, VIP, Elite)
- 📱 Social Feed
- 🔒 Security & Verification
